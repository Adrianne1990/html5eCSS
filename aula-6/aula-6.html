Aprendendo novas tags HTML
Aula 6

HTML é a linguagem de marcação padrão para criar páginas na Web. O HTML descrevea estrutura de uma página da Web e consiste em uma série de elementos. Os elementos HTML informam ao navegador como exibir o conteúdo.

CSS é a linguagem que descreve como os elementos HTML devem ser exibidos na tela, papel ou em outra mídia. O CSS economiza muito trabalho, pois pode controlar o layout de várias páginas da Web de uma só vez.

Javascript é a linguagem de programação o HTMl e da Web. Javascript pode alterar o conteúdo HTML e os valores dos atributos. Javascript pode ocultar e mostrar elementos HTML e muito mais.
